How it works

  1. Upload your GFS invoice CSV export
  2. System automatically parses all line items
  3. Review each item and check the ones that should be used by GSRP
  4. Recalculate the reconciliation, then approve the PDF report
  5. Report is emailed automatically with proper account codes

Allocation Rules

  • Items checked for GSRP use: 67% School Age / 33% GSRP
  • Items not checked: 100% School Age
  • Account Code (SA): 23.1.351.5610.200
  • Account Code (GSRP): 23.1.118.5110.000.3436.00629.0
